# 🏥 Healthcare Disease Pattern Discovery Using Machine Learning

## 📌 Project Overview

This project focuses on analyzing healthcare data to discover hidden disease patterns using Machine Learning techniques.  
The system helps in identifying risk factors and important health indicators from patient data.

It follows a complete end-to-end ML pipeline including preprocessing, model training, evaluation, and feature analysis.

---

## 🎯 Objectives

- Clean and preprocess healthcare dataset
- Perform Exploratory Data Analysis (EDA)
- Build multiple machine learning models
- Compare model performance
- Improve results using hyperparameter tuning
- Identify important health-related features
- Save the best trained model for future use

---

## 🧠 Machine Learning Models Used

- Logistic Regression  
- Decision Tree Classifier  
- Random Forest Classifier  
- Gradient Boosting Classifier  
- XGBoost Classifier  

---

## 📊 Key Features

- Age  
- BMI  
- Glucose Level  
- Cholesterol  
- HbA1c Level  
- Health Risk Score (Engineered Feature)

---

## ⚙️ Workflow

1. Data Loading  
2. Data Cleaning & Missing Value Handling  
3. Exploratory Data Analysis (EDA)  
4. Feature Engineering  
5. Data Preprocessing (Encoding & Scaling)  
6. Model Training  
7. Model Evaluation  
8. Hyperparameter Tuning  
9. Feature Importance Analysis  
10. Model Saving  

---

## 📈 Results

- Multiple ML models were trained and compared
- Random Forest was optimized using GridSearchCV
- Best model achieved improved accuracy after tuning
- Important health indicators were identified for prediction
